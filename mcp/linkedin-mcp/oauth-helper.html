LinkedIn OAuth Helper

This helper will get you a proper access token with your numeric member ID.

Step 1: Enter Your App Credentials

Step 2: Authorize with LinkedIn

Click the button below to open LinkedIn authorization. After authorizing, you'll be redirected with a code.

Required scope: w_member_social openid profile

Step 3: Enter the Authorization Code

After authorization, paste the code parameter from the redirect URL:

Manual cURL Command

If the button doesn't work, use this cURL command after getting the code:

curl -X POST https://www.linkedin.com/oauth/v2/accessToken \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=authorization_code" \
  -d "code=YOUR_CODE_HERE" \
  -d "redirect_uri=https://oauth.pstmn.io/v1/callback" \
  -d "client_id=77jwvf4x9tob6v" \
  -d "client_secret=YOUR_LINKEDIN_CLIENT_SECRET"
