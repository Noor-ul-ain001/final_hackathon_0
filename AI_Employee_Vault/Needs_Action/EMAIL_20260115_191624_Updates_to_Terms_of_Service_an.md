Hello Noor ul ain,

We’re writing to inform you about two upcoming updates related to Nebius Token Factory: an update to our Terms of Service, and the deprecation of legacy API keys.

Neither change affects your data ownership or current service usage, but some action is required regarding API keys (if you haven’t already).

Terms of Service update

We’ve updated our Terms of Service to better reflect how datasets are used across the platform, especially within Data Lab.

What’s new:

- The Terms of Service now explicitly define Datasets to include both data you upload and datasets you create within the platform (including via Data Lab).
- Datasets are formally included in Customer’s Content, confirming they remain owned by you.
- The Terms clarify how Datasets may be used across Data Lab, fine-tuning, distillation, and professional services you initiate.

What hasn’t changed:

- You retain all rights and ownership over your Datasets and other Customer’s Content.
- Your Datasets are used solely to provide the services you choose to run (for example, Data Lab workflows, fine-tuning, or distillation).
- Zero Data Retention (ZDR) remains fully under your control and can be configured in your user settings. When ZDR is enabled, inference logs are not stored and cannot be imported.

Legacy API keys deprecation

As part of ongoing improvements to security and access control, we will be deprecating legacy API keys on January 31, 2026.

What’s changing:

- Legacy API keys will no longer be supported after 31 January 2026.
- New granular API keys provide improved security, scoped permissions, better auditability, and safer key rotation.

How to migrate:

- Create new API keys in the API Keys section (https://email.nebius.com/e3t/Ctc/X+113/d38dPw04/VW1z397ZkCRqW5pb7hN4V_VLFW4bWRWk5JhsjSN3fD2D83prCCW7lCdLW6lZ3lnVsw4lw6KcFm_W6Qnb0P6KrYT_W6d2J9-4P-9yqW1BSxCM40dyB9W3ZCF9q9jZ8VvN7hbKgvyw_4DW59nh2W2n16gqW1mxfQw3kwqT7N1X_3lfhdGkzW2dY8p42wdp1YW54PDq83GzSDRW6sJyB937KyWRW1-kNhb20mmLkW4qdM7x50LfrgW6VrfWS3VGM4YN4lvnpTNxVKpW7Jqxyl7B8PjXN4m2_kGNc6n0W2n_hmM4nrprHW1rp02Y1qrTpTW5SX2XK680KvhW6XTRM8130hxRW46pm5j2Qv6nZW6pdlk46CMZzcdPQ1Vb04 ) of your Nebius console.
